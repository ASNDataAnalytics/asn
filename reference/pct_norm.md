# Function to calculate percentage change from first observation in vector

Function to calculate percentage change from first observation in vector

## Usage

``` r
pct_norm(x)
```

## Arguments

- x:

  Numeric vector

## Value

Numeric vector

## Examples

``` r
library(dplyr)
x <- 1:5
pct_norm(x)
#> [1] 0 1 2 3 4
```
